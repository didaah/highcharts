请先至此地址 http://www.highcharts.com/download，下载 highcharts，保存到模块目录下。
最终的目录结构如：
highcharts/highcharts.info
highcharts/highcharts.module
highcharts/README.txt
highcharts/highcharts/highcharts.js
highcharts/highcharts/modules/*
highcharts/highcharts/themes/*

若需要使用 highstock，下载后，将 highstock.js 保存到如下地址：
highcharts/highcharts/highstock.js
